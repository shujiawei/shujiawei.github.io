DOM扩展

选择符api

  1. document.querySelector();
  2. document.querySelectorAll();
  3. document.getElementsByClassName();

元素遍历

  1. childElementCount 返回子元素的个数，不包括文本节点和注释
  2. firstElementChild 返回第一个子元素 firstChild的元素版
  3. lastElementChild 返回最后一个子元素 lastChild的元素版
  4. previousElementSibling 返回前一个同辈元素 previousSibling的元素版
  5. nextElementSibling 返回后一个同辈元素 nextSibling的元素版

与类相关的扩充

  1. document.getElementsByClassName()
  2. classList
    2.1 add(value) 将给定的字符串添加到列表中，如果值已经存在就不添加了
    2.2 contains(value) 表示列表是否存在给定的值 存在返回true 不存在返回fasle
    2.3 remove(value) 从列表中删除给定的字符串
    2.4 toggle(value) 如果列表中存在给定的值就删除，不存在就添加
  3. 焦点管理
    3.1 document.activeElement 当前获得焦点的元素
    3.2 document.hasFocus() 确定文档是否获得了焦点
  4. 自定义数据属性
    4.1 dataset属性来访问自定义属性
  5. 插入标记
    5.1 innerHTML
    5.1.1 在读模式 innerHTML属性返回与调用元素的所有子节点(元素,注释和文本节点)
5.1.2 在写模式 innerHTML的值会被解析为DOM子树，替换调用元素原来的子节点
    5.2 outerHTML
    5.2.1 在读模式 outerHTML属性返回与调用元素的所有子节点(元素,注释和文本节点)
5.2.2 在写模式 outerHTML的值会被解析为DOM子树，完全替换调用元素
  6. scrollIntoView()
    6.1 scrollIntoView()可以在所有HTML元素上调用，通过滚动浏览器窗口或某个容器元素，调用元素就会出现在是口中
  7. children属性 只包含元素子节点
  8. contains()方法判断某个节点是不是另外一个节点的后代document.documentElement.contains(document. body)
  9. 插入文本
    9.1 innerText
    9.1.1 在读模式 innerText属性返回与调用元素的所有文本内容
9.1.2 在写模式 innerText的值会替换容器元素的所有内容
    9.2 outerText
    9.2.1 在读模式 outerText属性返回与调用元素的所有文本内容
9.2.2 在写模式 outerText的的值会替换容器元素

DOM文档对象模型

DOM描绘了一个层次化的节点数

文档节点是每个文档的根节点，文档节点只有一个子节点就是html元素。

Node类型

每个节点都有一个nodeType类型

1: 元素节点
3: 文本节点
6: 注释节点

每个节点都有一个childNodes属性,其中保存着一个NodeList对象.NodeList是一种类数组对象，用于保存一组有序的节点，可以通过位置来访问这些节点。

如何访问childNodes?

  1. 直接通过索引来访问
    var firstChild = someNode.childNodes[0]
  2. 通过item()来访问
    var fisrtChild = someNode.childNodes.item(0)

延伸知识?(如何把类数组转化为数组)

  1. Array.propotype.slice.call(nodeList, 0);

  2. 手动枚举所有成员
    var arr = [];
    for(var i = 0; i < nodeList.length; i++) {

    arr.push(nodeList[i])

    }

  3. var arr = Array.from(nodeList);

  4. var arr = […nodeList];

介绍下firstChild,lastChild,previousSibling, nextSibling

firstChild = parentNode.childNodes[0];

lastChild = parentNode.childNodes[parentNode.childNodes.length - 1];

previousSibing 获取当前节点的上一个兄弟节点

nextSibling 获取当前节点的下一个兄弟节点

值得一提的是ownerDocument，该属性指向表示整个文档的文档节点，任何节点都不能同时存在两个或者更多的文档中。

如何操作节点？

appendChild(), insertBefore(), replaceChild(), removeChild(), cloneNode()

appendChild(要插入的节点) 用于向childNodes列表末尾添加一个节点

insertBefore(要插入的节点，作为参照的节点) 把插入节点放到参照节点前面

replaceChild(要插入的节点, 要替换的节点) 把要替换的节点更换为插入节点

removeChild(移除的节点) 把该节点从childNodes中移除

cloneNode(克隆的节点, 布尔值(true为深拷贝, false为浅拷贝)) 返回一个完全相同的副本

Document类型

  1. nodeType的值为9
  2. nodeName的值为#document
  3. nodeValue为null
  4. parentNode为null
  5. ownerDocument为null

文档的子节点

  1. document.documentElement 取得对html的引用
  2. document.body 取得对body的引用

文档的信息

  1. document.URL 地址栏URL
  2. document.domain 页面域名
  3. document.referrer 保存者链接到当前页面的那个页面的URL

查找元素

  1. document.getElementId(元素的id)
  2. document.getElementsByTagName(标签名称) 返回的集合是HTMLCollection对象

文档写入

  1. document.write()

Element类型

  1. nodeType的值为1
  2. nodeName的值为元素的标签名
  3. nodeValue为null
  4. parentNode可能为Document或Element

访问元素的标签名称可以用nodeName属性或者tagName属性

HTML元素

  1. id元素在文档中唯一标识符
  2. title有关元素的附加说明信息，一般通过工具提示条显示出来
  3. lang元素内容的语言代码，很少使用
  4. dir语言的方向，值为”ltr”(left to right, 从左到右)或”rtl”(right to left,从右到左)
  5. calssName与元素的class特性对应，即为元素指定的CSS类

上述的属性都可以用来取得或修改相应的特性值

var div = document.getElementId(‘#div’);
div.id = ‘div’;

HTML特性

  1. someNode.getAttribute(属性名)
  2. someNode.setAttribute(‘属性名’, ‘属性值’)
  3. someNode.removeAttribute(属性名)

创建元素

document.createElement(标签名)

var div = document.createElement(‘div’)
div.id = ‘div’;
document.body.appendChild(div)

DocumentFragment类型

  1. nodeType为11
  2. nodeName为#document-fragment
  3. nodeValue为null

JSON

JSON是一种数据格式，不是一种编程语言
JSON字符串必须是双引号包裹，不能是单引号
JSON不允许添加注释

语法

  1. 简单值：字符串,数值,布尔值和null但不支持undefined
  2. 对象：对象作为一种复杂数据类型，表示的是一组无序的键值对儿。而每对键值对儿的值可以是简单值，也可以是复杂数据类型的值
  3. 数组：数组也是一种复杂数据类型，表示一组有序的值的列表，可以通过数值索引来访问其中的值。数组的值也可以是任意类型–简单值，对象，数组

解析与序列化

  1. JSON.stringigy() 将JSON对象转化为JSON字符串
  2. JSON.parase() 将JSON字符串转化为JSON对象

下面来讲解这两个方法

JSON.stringigy(value[, replacer [, space]])

value: 序列化的对象 必填
replacer: 过滤器，可以是一个数组，也可以是一个函数
space: 表示是否在JSON字符串保留缩进

用处

  1. 把对象转化为字符串
  2. 把字符串转化为对象
  3. 进项深拷贝(对象属性没有方法)
